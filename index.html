HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The biggest heading, h1, should only appear once on each page.
  • Pro Tip: When starting from scratch with no existing code, you can easily start from a template by creating an index.html file, press Shift+1 to create an excalamtion point then press tab. This will create a basic HTML template.
  • Pro Tip: You can easily add a link to CSS by going under title, typing link and hitting enter on the tooltip that pops up. This will create a formatted link for you to point to the CSS file.
  • Pro Tip: To easily create a list item or unordered list or anything in HTML, really, just start typing it and press enter when it is selected on the tooltip. Code VS will insert the element for you and make it quicker to modify.
  • Pro Tip: To easily create a javascript link in your html file, type script:src and press tab, Code VS will create and format a script element for you, then you just point it to the javascript file (Enter it after the src="").
  • Pro Tip: To easily create classes in VS Code, use dot and type in your classes then press tab, for example .emoji.closed.active.
  • Pro Tip: You can press Shift+Option+Down arrow to clone the same line below the current one.
  • Pro Tip: Command+Shift+C to open console in Chrome

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Don't forget to link the external css sheet to your HTML file.

Git

  • git status: checks what branch we are currently on
  • git checkout branch-name: switches git to the branch you enter
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Pro-Tip: When working with dataset attributes, you can access any attribute with "data-" in front by using element.dataset.* Replace * with whatever attribute you want to access.
